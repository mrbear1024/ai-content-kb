# Staging

All unreviewed AI-generated prose, course control files, relationship sidecars, and bulk transformations start here.

Review candidates for accuracy, provenance, duplication, ownership, and intended use before promoting or discarding them.

- `drafts/`: article and general prose drafts.
- `course-drafts/`: course outlines and control-file drafts.
- `wiki/`: candidate concept, entity, map, and source-note pages.
- `links/`: candidate relationship sidecars.
